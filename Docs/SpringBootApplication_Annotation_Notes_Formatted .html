✅ @SpringBootApplication Annotation – Complete Notes (Spring Boot 3 / Java 17)

1️⃣ What is @SpringBootApplication?

@SpringBootApplication is the main entry-point annotation in a Spring Boot application.

  • Enables auto-configuration
  • Enables component scanning
  • Enables configuration support

It is a meta-annotation, combining multiple annotations into one.

2️⃣ Where is it Used?

package com.example.demo;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class DemoApplication {

    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
  • Bootstraps the Spring container
  • Starts embedded server (Tomcat by default)
  • Triggers auto-configuration
  • Performs component scanning

3️⃣ What Does @SpringBootApplication Contain?

@Configuration
@EnableAutoConfiguration
@ComponentScan

3.1️⃣ @Configuration

@Configuration
public class AppConfig {

    @Bean
    public MyService myService() {
        return new MyService();
    }
}

Marks class as source of bean definitions.

3.2️⃣ @EnableAutoConfiguration

Enables Spring Boot’s auto-configuration mechanism.


    org.springframework.boot
    spring-boot-starter-web

3.3️⃣ @ComponentScan

Scans for:

  • @Component
  • @Service
  • @Repository
  • @Controller
  • @RestController
com.example.demo
    ├── DemoApplication.java
    ├── controller
    ├── service
    ├── repository

4️⃣ Internal Structure (Simplified)

@Target(ElementType.TYPE)
@Retention(RetentionPolicy.RUNTIME)
@Documented
@Inherited
@SpringBootConfiguration
@EnableAutoConfiguration
@ComponentScan
public @interface SpringBootApplication {
}

5️⃣ Important Attributes

Change Scan Base Package

@SpringBootApplication(scanBasePackages = "com.mycompany")

Exclude Auto Configuration

@SpringBootApplication(
    exclude = {DataSourceAutoConfiguration.class}
)

6️⃣ Startup Flow

main()
   ↓
SpringApplication.run()
   ↓
Creates ApplicationContext
   ↓
Performs Component Scan
   ↓
Applies Auto Configuration
   ↓
Registers Beans
   ↓
Starts Embedded Server

7️⃣ Real-World Example

@RestController

    }

8️⃣ Best Practices

  • Place main class at root package
  • Avoid unnecessary scanBasePackages
  • Use exclude carefully
