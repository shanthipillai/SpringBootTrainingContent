📘 Thymeleaf – Complete Notes

🔥 What is Thymeleaf?

Thymeleaf is a server-side Java template engine used to build dynamic web pages in Spring Boot.

🧩 Why Thymeleaf?

  • Natural templating
  • Easy integration with Spring Boot
  • No scriptlets
  • Supports data binding, iteration, conditions

⚙️ Dependency

<dependency> <groupId>org.springframework.boot</groupId> <artifactId>spring-boot-starter-thymeleaf</artifactId> </dependency>

📁 Folder Structure

src/main/resources/templates/

🔗 Controller Example

@Controller public class MyController { @GetMapping("/home") public String home(Model model) { model.addAttribute("name", "Shanthi"); return "home"; } }

🧾 Display Data

<p th:text="${name}"></p>

🧩 Core Attributes

  • th:text
  • th:utext
  • th:if / th:unless
  • th:each
  • th:value
  • th:href
  • th:src

🧠 Expressions

  • ${} - Variable
  • *{} - Selection
  • @{} - Links
  • #{} - Messages

🧾 Forms

<form th:action="@{/register}" th:object="${user}" method="post"> <input th:field="*{name}"> </form>

🔁 Controller Handling

@PostMapping("/register") public String register(@ModelAttribute("user") User user) { return "success"; }

🧩 List Display

<tr th:each="u : ${users}"> <td th:text="${u.id}"></td> </tr>

⚠️ th:object vs ${}

FeatureUsage
${}Normal variable
th:objectForm binding
*{}Form fields

❗ Common Mistakes

  • Forgetting th:

📘 Thymeleaf Expression Types with Examples

🔹 ${} → Variable Expression

Used to access model data (attributes from controller)

model.addAttribute("name", "Shanthi");
model.addAttribute("age", 25);

<p th:text="${name}"></p>
<p th:text="${age}"></p>

Output:
Shanthi
25

🔹 *{} → Selection Expression

Used with th:object (form binding). Refers to current object.

model.addAttribute("user", new User("Shanthi", "shanthi@gmail.com"));

<form th:object="${user}">
Name: <input type="text" th:field="*{name}" />
Email: <input type="email" th:field="*{email}" />
</form>

🔹 @{} → URL Expression

<a th:href="@{/home}">Home</a>
<a th:href="@{/user/{id}(id=101)}">View User</a>
<a th:href="@{/search(name='Shanthi', age=25)}">Search</a>

🔹 #{} → Message Expression (i18n)

welcome.message=Welcome to Spring Boot
login.title=Login Page

<h1 th:text="#{welcome.message}"></h1>
<p th:text="#{login.title}"></p>

🎯 Quick Summary

ExpressionPurposeExample
${}Model data${name}
*{}Form object*{email}
@{}URL/link@{/home}
#{} Messages#{welcome.message}

💡 Teaching Tip

  • ${} → Outside form
  • *{} → Inside form
  • @{} → Navigation
  • #{} → Properties/messages
