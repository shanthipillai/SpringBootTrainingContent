📘 JSTL (JavaServer Pages Standard Tag Library)

🔥 What is JSTL?

  • JSTL is a standard tag library for JSP
  • Avoids Java scriptlets
  • Provides tags for logic, loops, conditions

🧩 Why JSTL?

❌ Without JSTL

<% for(int i=0;i<5;i++){ out.println(i); } %>

✅ With JSTL

${i}
