🌩️ Spring Cloud – Complete Notes

🎯 What is Spring Cloud?

Spring Cloud is a framework used to build distributed systems / microservices architecture.

  • Configuration management
  • Service discovery
  • Load balancing
  • Fault tolerance
  • API routing

🏗️ Microservices Architecture

Flow:

  • Client → API Gateway
  • Gateway → Microservices
  • Services register with Eureka
  • Config fetched from Config Server

1️⃣ Config Server

Centralized configuration for all services

<dependency> <groupId>org.springframework.cloud</groupId> <artifactId>spring-cloud-config-server</artifactId> </dependency> @EnableConfigServer @SpringBootApplication spring: cloud: config: server: git: uri: https://github.com/your-repo/config-repo
  • No hardcoding configs
  • Change config without redeploy
  • Common config

2️⃣ Eureka Server

<dependency> <groupId>org.springframework.cloud</groupId> <artifactId>spring-cloud-starter-netflix-eureka-server</artifactId> </dependency> @EnableEurekaServer server: port: 8761 eureka: client: register-with-eureka: false fetch-registry: false

URL: http://localhost:8761

3️⃣ Eureka Client

spring: application: name: account-service eureka: client: service-url: defaultZone: http://localhost:8761/eureka/

4️⃣ Feign Client

@FeignClient(name = "card-service") public interface CardClient { @GetMapping("/cards/account/{accountId}") List<Card> getCards(@PathVariable int accountId); }
  • No boilerplate
  • Clean code
  • Works with Eureka

5️⃣ API Gateway

spring: cloud: gateway: routes: - id: account-service uri: lb://account-service predicates: - Path=/accounts/**
  • Routing
  • Authentication
  • Logging
  • Rate limiting

6️⃣ Load Balancing

@LoadBalanced @Bean public RestTemplate restTemplate() { return new RestTemplate(); }

http://account-service/accounts

7️⃣ Circuit Breaker

@CircuitBreaker(name = "cardService", fallbackMethod = "fallback") public List<Card> getCards(int accountId) { return cardClient.getCards(accountId); } public List<Card> fallback(int accountId, Exception e) { return new ArrayList<>(); }

🔁 Flow Summary

  • Client → Gateway
  • Gateway → Service
  • Service → Config Server
  • Feign calls
  • Circuit breaker handles failure

⚙️ Annotations Cheat Sheet

  • @EnableConfigServer
  • @EnableEurekaServer
  • @EnableDiscoveryClient
  • @EnableFeignClients
  • @CircuitBreaker
